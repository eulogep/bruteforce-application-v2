## Architecture Technique

### Frontend
- **Framework :** React 19.1.0 avec Vite
- **UI :** Tailwind CSS + shadcn/ui
- **Déploiement :** Site statique optimisé

### Backend
- **Framework :** Flask 3.1.1 avec Python 3.11
- **Base de données :** SQLite avec SQLAlchemy
- **Fonctionnalités :** Threading, CORS, API REST

### Communication
- **API REST :** Communication entre frontend et backend
- **CORS :** Configuré pour permettre les requêtes cross-origin
- **Format :** JSON pour tous les échanges de données
